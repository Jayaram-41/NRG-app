# NRG Foods Authentication Application

A full-stack authentication application built with Angular frontend and .NET 8 backend.

## Project Overview

This application provides user authentication functionality including login, registration, and dashboard features.

## Project Structure

- **Frontend**: Angular application in `nrg-auth-app/`
- **Backend**: .NET 8 Web API in `NrgAuthApi/`
